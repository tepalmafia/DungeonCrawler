# 에셋 크레딧

## 아트 (캐릭터·몬스터·함정·소품)

- **직접 제작한 픽셀 아트**. 외부 에셋·이미지 생성 도구 없이, 저해상 픽셀 그리드를
  코드로 저작하고(팔레트 + 프리미티브 + 자동 외곽선 + 수직 음영), 런타임에
  니어리스트 확대(`imageSmoothingEnabled=false`)로 선명하게 렌더.
- 정의 위치: `WebPrototype/index.html`의 `PXPAL`(팔레트) / `PXSPR`(스프라이트 그리드) / `pixelSprite()`.
- 스프라이트: 헌터(아처)·멧돼지·사슴·화염 곰 + 속박 덫·마비 룬·가시 지대·미끼.

## 사운드

- WebAudio로 런타임 절차 생성 — 외부 파일 없음.

전 에셋이 리포지토리 내에서 코드로 생성되며, 외부 라이선스 의존성이 없습니다.
