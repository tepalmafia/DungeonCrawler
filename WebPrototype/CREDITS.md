# 에셋 크레딧

## 아트 (캐릭터·몬스터·함정·소품)

- **출처**: [game-icons.net](https://game-icons.net)
- **라이선스**: [CC BY 3.0](https://creativecommons.org/licenses/by/3.0/)
- **제작자**: Lorc, Delapouite 외 (game-icons.net 기여자)
- **사용 방식**: 원본 SVG 아이콘(단색 실루엣, 512 그리드)의 경로 데이터를 인라인으로 임베드하고,
  런타임에 세로 그라디언트(입체 음영)·외곽선 스트로크·드롭섀도우·상단 하이라이트를 코드로 입혀
  게임 스프라이트로 사용. (`WebPrototype/index.html`의 `GICON` / `iconSprite` / `iconDecal`)

사용 아이콘: `boar`, `deer`, `archer`(헌터), `polar-bear`(화염 곰),
`mantrap`(속박 덫), `lightning-arc`(마비 룬), `spikes-full`(가시 지대), `meat`(미끼).

## 사운드

- WebAudio로 런타임 절차 생성 — 외부 파일 없음.
