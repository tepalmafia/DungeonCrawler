# 헌터의 의뢰 (가제) — 게임 기획서

> 당신은 몬스터 사냥 청부업자. 잡기 어려운 몬스터 1마리의 의뢰를 받아, 정보를 모으고 함정·장비·스킬을 준비한다.
> 사냥이 시작되면 손을 뗀다 — 준비한 만큼만 결과가 나온다.

**장르**: 세팅 중심 자동전투 × 로그라이트 몬스터 사냥
**시점**: 2D 탑다운 (기존 DungeonCrawler Unity 2D URP 프로젝트 기반)
**한 판 길이**: 15~25분 (준비 무제한 + 사냥 3분)

---

## 1. 핵심 루프

한 판 = 의뢰 → 정찰 → 준비 → 사냥 → 정산.

1. **의뢰 선택** — 몬스터 3종 중 1개 선택. 난이도·보상·알려진 정보량이 다름
2. **정찰 (선택)** — 준비 골드를 써서 몬스터의 약점·행동 패턴·숨은 기믹을 공개.
   알아낸 정보는 **실패해도 도감에 영구 기록** → 재도전이 로그라이트답게 쉬워짐
3. **준비** — 공용 자원 "준비 골드"를 4곳에 분배 (전부 살 수 없게 설계, §7 경제 참고)
   - **장비**: 무기(피해·속성)·방어구(내성). 몬스터 약점에 맞추는 게 핵심
   - **아이템**: 소모품 — 미끼(유인), 폭탄, 연막(도주 방지), 회복 물약
   - **함정**: 사냥터 맵에 직접 배치. 위치 선정이 플레이어의 실력 요소
   - **스킬(감빗)**: 자동전투 중 발동할 스킬 + 발동 조건 설정 (§5)
4. **사냥 (자동전투, 제한 시간 3분)** — 플레이어는 관전 + 긴급 지시 카드 3장(§3)만 개입 가능
5. **정산** — 남은 시간·부위 파괴 등으로 보상 배율 (§7). 실패해도 도감과 위로금은 유지

### 확정된 핵심 결정 사항

| 항목 | 결정 |
|---|---|
| 준비 시간 | 무제한. 플레이어가 준비를 마치면 사냥 시작 |
| 시간 압박 | 사냥 페이즈 3분 제한 — 시간 내 못 잡으면 몬스터 도주 |
| 전투 방식 | 자동전투 + 세팅 중심 (감빗 시스템) |
| 사냥 중 개입 | 긴급 지시 카드 3장 |
| 헌터 사망 | 즉사 아님 — 10초 기절 후 부활 (시간 손실이 페널티) |
| 감빗 슬롯 | 3개 시작 → 랭크업으로 최대 8개 |
| 은신 중 감빗 | 완전 정지, 생존 계열(★)만 작동 |
| 행동 성장 | 행동 자체는 고정, 수치 성장은 장비 스탯으로만 |
| 반복 구조 | 로그라이트 — 판 내 자원 리셋, 도감·랭크·명예 pt만 이월 |

---

## 2. 사냥 페이즈 수치

- **제한 시간 3:00** 기준 흐름:
  `0:00~0:20 경계(몬스터가 맵 배회, 미끼·함정 유인 타임) → 교전 → HP 조건 충족 시 격노 → 잔여 30초 or HP 조건 시 도주 시도`
- **도주 시도**: 몬스터가 맵 가장자리 탈출구(2곳)로 이동. 몬스터별 지정 수단으로 저지 가능.
  저지 실패 = 사냥 실패
- **도주 저지의 일관 규칙**: 모든 도주는 "몬스터별 지정 수단으로 1회 저지 가능,
  저지 시 도주 게이지 리셋"으로 통일. 몬스터마다 저지 수단만 달라짐
- **헌터 기절**: HP 0 → 10초 기절 후 부활. 기절 시 몬스터는 교전을 풀고 경계로 복귀
  (다시 유인할 기회를 주는 리셋. 단 타이머는 계속 흐름)

## 3. 긴급 지시 카드

관전 중 유일한 개입 수단. "자동전투의 빈틈을 메우는 보험"이지 조작 수단이 아님.

- **사용 방식**: 사냥 시작 전 카드 풀에서 3장 선택해 입장. 전투 중 클릭 1번으로 발동
  (쿨타임 없음, 각 1회용)
- **설계 의도**: 카드 선택 = "이 몬스터 상대로 뭐가 잘못될 수 있지?" 예측.
  도감 정보가 많을수록 카드 선택도 정확해짐
- **보상 연동**: 미사용 1장당 보상 배율 +10% — 완벽한 준비에 대한 인센티브

| 카드 | 효과 | 쓰는 상황 |
|---|---|---|
| 후퇴 신호 | 헌터가 3초간 무적으로 이탈 | 격노 타이밍을 잘못 읽었을 때 |
| 함정 재가동 | 이미 발동된 함정 1개 재설치 | 함정이 헛발동했을 때 |
| 집중 사격 | 5초간 공격속도 2배 | 막판 시간이 모자랄 때 |
| 긴급 미끼 | 몬스터 어그로를 지정 위치로 강제 이동 | 몬스터가 함정 지대를 피해갈 때 |

카드 풀은 명예 pt로 확장 (§7 메타 경제).

## 4. 몬스터 설계

### 4.1 설계 원칙: 열쇠와 자물쇠

몬스터마다 **준비 없이는 못 잡는 기믹**을 1개 이상 부여한다.
"장비냐 함정이냐 스킬이냐"가 아니라 **몬스터가 요구하는 조합**을 읽어내는 게임.

### 4.2 공통 상태머신

모든 몬스터가 공유하는 골격 (기존 `EnemyAI` 확장으로 구현):

```
경계(Wander) → 교전(Combat) → 격노(Enrage) → 도주(Flee)
                    ↕
                피격/함정(Stunned)
```

- **경계**: 지정 경로 배회. 헌터 발견(시야 반경) 또는 피격 시 교전 진입.
  **미끼에 반응하는 유일한 상태** — 교전 중엔 미끼 효과 절반
- **교전**: 개체별 공격 패턴 루프 실행
- **격노**: 전이 조건 충족 시 진입. 스탯 변화 + 패턴 변화
- **도주**: 탈출구로 이동, 이동 중 공격 안 함. 속박 계열로 저지 시 교전 복귀 + 게이지 리셋
- **함정 걸림**: 모든 행동 중단. 이 상태에서 받는 피해 **2배** (강공격 감빗의 존재 이유)

### 4.3 초기 로스터 5종

난이도 곡선: 1번은 실패 불가(시스템 학습), 2~4번은 각각 "함정 배치 / 정보·빌드 선택 /
보험 설계"라는 서로 다른 근육을 단련, 5번은 종합 시험. 이 5종이 로그라이트 1사이클(랭크 1→2).

| # | 몬스터 | 기믹 | 요구 준비 (열쇠) |
|---|---|---|---|
| 1 | 멧돼지 왕 | 없음 (튜토리얼) | 기본 장비면 충분 |
| 2 | 질풍 사슴 | 도주형: 이속 2배, 1:30부터 도주 | 속박 덫 + 미끼 동선 |
| 3 | 화염 곰 | 격노형: HP 50%↓ 공격력 3배 | 화염 내성 or 폭딜로 격노 스킵 |
| 4 | 그림자 살쾡이 | 은신형: 주기적 투명화 | 섬광 트랩으로 강제 노출 |
| 5 | 서리 독수리 | 비행+도주 조합 (사이클 보스) | 그물 발사기 + 속박 덫 |

#### 1. 멧돼지 왕 — 튜토리얼

- **스탯**: HP 300 / 이속 보통 / 공격력 낮음
- **교전 패턴**: `2초 조준 → 직선 돌진 (회피 쉬움) → 3초 경직` 반복. 경직이 명백한 딜 타임
- **격노**: HP 30% 이하. 조준 시간 2초 → 1초 단축뿐 (위협 미미)
- **도주**: 없음
- **가르치는 것**: 돌진 경로에 함정 → 함정 걸림 = 피해 2배 → "속박 중 강공격" 감빗의 가치

#### 2. 질풍 사슴 — 도주형

- **스탯**: HP 250 / 이속 2배 / 공격 안 함 (뿔치기 반격만)
- **핵심 기믹**: 교전 중에도 거리를 유지하며 도망다님 → 함정 없이는 딜 자체가 불가
- **추가 상태 — 질주(Sprint)**: 5초마다 1.5초간 이속 3배로 방향 전환.
  함정 지대를 우회할 수 있어 함정을 **부채꼴로** 깔아야 함 (배치 실력이 드러나는 지점)
- **도주 전이**: 1:30 경과 **or** HP 40% 이하 — 폭딜로도 도주 페이즈를 피할 수 없음.
  저지 수단(속박 덫·연막·긴급 미끼)을 준비했는지 검증
- **정답 준비**: 미끼로 경계 상태에서 함정 지대로 유인 → 속박 → 폭딜

#### 3. 화염 곰 — 격노형

- **스탯**: HP 500 / 이속 느림 / 공격력 높음
- **교전 패턴**: `앞발 휘두르기(근접) → 화염 브레스(전방 부채꼴, 화상 도트)` 루프
- **격노**: HP 50% 이하. **공격력 3배 + 화염 오라**(근접 시 초당 도트)
- **격노 해제 (숨은 정보, 정찰로만 공개)**: 격노 15초 지속 후 5초간 **탈진**(방어력 0).
  알면 "격노를 버티고 탈진에 폭딜", 모르면 도망만 다니게 됨 — 도감 가치의 대표 사례
- **도주**: 없음
- **정답 준비 두 갈래** (양쪽 다 성립하게 밸런싱):
  (a) 화염 내성 + 회복 감빗으로 버티며 탈진 폭딜
  (b) 함정 몰빵으로 격노 진입 전에 HP 50%→0 폭딜 스킵

#### 4. 그림자 살쾡이 — 은신형

- **스탯**: HP 350 / 이속 빠름 / 공격력 중간
- **추가 상태 — 은신(Stealth)**: 교전 진입 후 10초마다 5초간 투명.
  투명 중 타겟팅 불가(감빗 완전 정지, 생존 계열 ★만 작동), 위치는 발자국 이펙트로만 힌트
- **은신 해제 시 기습**: 은신 직전 헌터 위치를 기억하고 등 뒤에서 2배 피해 일격
  → 감빗 "은신 직전 → 위치 이탈"이 카운터
- **섬광 트랩 상호작용**: 투명 중 섬광 범위 진입 시 강제 노출 + 3초 기절
- **도주**: HP 25% 이하에서 **은신 상태로** 탈출구로 달림.
  탈출구에 섬광을 미리 깔았는지가 마지막 검증 — "함정은 교전용만이 아니다"
- **정답 준비**: 섬광 트랩(교전용 + 탈출구 보험용), 위치 이탈 감빗, 남는 골드로 방어구

#### 5. 서리 독수리 — 비행+도주 조합 (사이클 보스)

- **스탯**: HP 450 / 비행 / 공격력 중간 (급강하 공격)
- **추가 상태 — 활공(Airborne)**: 기본 상태. **지상 함정 전부 무효, 근접 공격 불가.**
  원거리 무기 or 신규 해금 장비 "그물 발사기"로만 대응
- **패턴 루프**: `활공(얼음깃털 투척) → 15초마다 급강하 → 지상 3초 체류 → 재이륙`.
  지상 체류 3초가 유일한 함정 타이밍 — 급강하 낙하지점(헌터 현재 위치 기준) 예측 배치가 핵심
- **그물 발사기** (감빗: "몬스터가 지상에 있으면 → 그물"): 명중 시 10초 비행 불가
  → 속박 덫 연계 → 지상전 공식이 그대로 적용
- **격노**: HP 40% 이하. 급강하 주기 15초 → 8초. 위험해지지만 지상 체류도 잦아짐 (리스크-리턴)
- **도주**: 잔여 40초에 고도 상승 후 이탈 — **그물 명중으로만 저지 가능**.
  그물을 교전에 다 쓰면 못 막음 → 탄약 관리(3발)라는 마지막 레이어
- **요구 조합**: 낙하지점 예측 배치(2번) + 격노 리스크 관리(3번) + 도주 저지 보험(4번) + 그물 탄약 관리(신규)

## 5. 감빗 시스템 (조건-행동 규칙)

### 5.1 규칙 구조

- 감빗 1줄 = **조건(IF) + 행동(THEN)**. 슬롯은 랭크에 따라 3 → 8개
- **우선순위 방식**: 위에서 아래로 검사, 조건을 만족하는 **가장 위의 줄 하나만 실행**.
  아무 줄도 발동 안 하면 기본 행동(평타 + 추적) — 감빗 없이도 싸우긴 싸움
- **쿨타임은 행동에 붙음**: 연속 발동 방지 + "회복약 도배" 악용 방지.
  같은 행동의 중복 배치는 허용 (쿨타임 공유라 악용 불가)
- **조건·행동은 랭크·도감 보상으로 해금**되는 지식 자산 — 준비 골드를 안 먹으므로
  "돈 없이도 쌓이는 실력"이라는 위치

### 5.2 조건(IF) 목록 — 16종

**기본 제공 (랭크 1)**

| 조건 | 설명 |
|---|---|
| 항상 | 무조건 참. 최하단 기본 행동 지정용 |
| 내 HP < X% | X는 30/50/70 중 선택 |
| 몬스터 HP < X% | X는 25/50 중 선택 |
| 몬스터가 함정에 걸림 | 속박·기절 등 함정 걸림 상태 전체 |

**랭크 2 해금 — 페이즈 읽기**

| 조건 | 설명 |
|---|---|
| 몬스터가 격노 상태 | 격노 진입~해제 동안 참 |
| 몬스터가 도주 시도 중 | 도주 상태 진입 시 참 |
| 몬스터가 탈진/경직 상태 | 탈진, 돌진 후 경직 등 딜 타임 전반 |
| 남은 시간 < 60초 / 30초 | 두 버전 별도 조건 |

**랭크 3 해금 — 위치와 거리**

| 조건 | 설명 |
|---|---|
| 몬스터와 거리 > 근접 사거리 | 원거리 전환·추격 판단용 |
| 몬스터가 내 함정 반경 X 안 | "함정 근처로 왔을 때 유인" 연계용 |
| 내가 몬스터의 공격 범위 안 | 조준선·브레스 예고 범위 감지 |
| 몬스터가 공중에 있음 | 서리 독수리 대응. 그물 발사기와 세트 |

**도감 보상 해금 — 몬스터별 특화** (해당 몬스터 정보 3개 이상 기록 시)

| 조건 | 출처 몬스터 |
|---|---|
| 몬스터가 은신 직전 (예비 동작 감지) | 그림자 살쾡이 |
| 몬스터가 돌진 조준 중 | 멧돼지 왕 |
| 몬스터가 급강하 낙하 중 | 서리 독수리 |
| 격노 해제까지 5초 이내 | 화염 곰 |

특화 조건 = "이 몬스터를 연구한 사람만 짤 수 있는 규칙". 도감이 전투력으로 환산되는
두 번째 경로 (첫 번째는 함정 배치 화면의 동선 표시).

### 5.3 행동(THEN) 목록 — 14종

★ = 은신 중에도 작동하는 생존 계열.

**전투 계열**

### 5.4 모범 규칙표 예시

**화염 곰 (버티기 빌드, 슬롯 5)** — 격노 중엔 4번이 잡아서 도망 다니다,
탈진이 뜨는 순간 2번이 우선권을 뺏어 폭딜. 우선순위 배치가 곧 전략:

1. 격노 해제까지 5초 이내 → 필살기
2. 몬스터가 탈진 상태 → 강공격
3. 내 HP < 50% → 회복 물약
4. 몬스터가 격노 상태 → 거리 벌리기
5. 항상 → 평타 지속

**그림자 살쾡이 (슬롯 5)** — 은신 중엔 1~2·4~5번이 얼어붙고 3번(★)만 살아있는 상태가
발자국 연출과 함께 긴장감을 만듦:

1. 몬스터가 은신 직전 → 위치 이탈
2. 몬스터가 함정에 걸림 → 강공격
3. 내 HP < 30% → 회복 물약
4. 몬스터와 거리 > 근접 사거리 → 원거리 사격
5. 항상 → 평타 지속

### 5.5 밸런스 안전장치

- 조건 16 × 행동 14 = 이론상 224조합이지만 쿨타임·소모품 재고가 실질 상한
- "회복 물약 도배"는 물약 가격(골드 경쟁)이, "구르기 무적"은 쿨타임 4초의 빈틈이 막음
- 행동 자체는 성장하지 않음 — 수치 성장은 장비 스탯으로만 (시스템 복잡도 억제)

## 6. 화면 구성 (2D 탑다운)

1. **의뢰 게시판** — 몬스터 3종 카드. 실루엣 + 알려진 정보만 표시 (도감 미기록은 "???")
2. **준비 화면** — 좌: 골드/인벤토리, 중앙: 장비 슬롯 + 감빗 규칙표(드래그로 우선순위 정렬,
   줄마다 ON/OFF 토글), 우: 긴급 카드 3슬롯
3. **함정 배치 화면** — 사냥터 맵을 미리 보며 드래그 배치. 몬스터 예상 동선이 도감
   정보량에 따라 흐릿하게/선명하게 표시 — 정찰의 가치가 시각적으로 드러나는 지점
4. **사냥 화면** — 탑다운 관전. 상단: 타이머 + 몬스터 페이즈 게이지, 하단: 긴급 카드 3장,
   우측: 감빗 발동 로그(발동 줄 하이라이트, 예: `③ HP<50% → 회복 물약`) —
   내 규칙표가 일하는 걸 보여주는 게 관전 재미의 핵심
5. **정산 화면** — 배율 계산 연출 → 도감 갱신 → 랭크 경험치 + **감빗 줄별 발동 횟수 통계**
   ("4번 줄이 한 번도 안 터졌네" 복기 가능 — 세팅 개선의 근거를 주는 핵심 피드백 루프)

## 7. 경제 밸런스

### 7.1 설계 원칙

1. **준비 골드는 매 판 리셋** — 남은 골드 소멸. "아껴서 이월"이 아니라 "이번 판에 다 태울지"만 고민
2. 이월되는 건 **명예 포인트**(메타 화폐)와 **도감·랭크**뿐 — 판 내 경제와 메타 경제 분리
3. 목표 구매력: **"필요한 것의 약 70%"만 살 수 있는 예산** → 나머지 30%를 뭘 포기할지가 매 판의 의사결정
4. **장비는 판 내 대여** (판 종료 시 반납) — 소유형이면 3~4판 후 살 게 없어져 준비 페이즈가 죽음

### 7.2 상점 가격표

**장비 (대여)**

| 품목 | 가격 | 효과 |
|---|---|---|
| 무기 강화 Lv1 / Lv2 | 200 / 400G | 평타·강공격 피해 +25% / +50% |
| 속성 부여 (화염/냉기/전격) | 150G | 약점 적중 시 추가 +30% |
| 방어구 Lv1 / Lv2 | 200 / 350G | 받는 피해 -20% / -35% |
| 속성 내성 | 150G | 해당 속성 피해 -50%, 도트 무효 |
| 그물 발사기 (랭크 2 해금) | 300G | 그물 3발. 감빗 행동 "그물 발사" 개방 |

**함정**

| 품목 | 가격 | 효과 |
|---|---|---|
| 속박 덫 | 200G | 4초 속박. 도주 저지 가능 |
| 전격 트랩 | 150G | 2초 기절 + 피해 |
| 섬광 트랩 | 150G | 은신 강제 해제 + 3초 기절 |
| 낙석 트랩 | 250G | 큰 피해 단발. 속박 연계 전제 |
| 가시 지대 | 100G | 밟는 동안 도트 + 이속 -30% |

**소모품**

| 품목 | 가격 | 비고 |
|---|---|---|
| 회복 물약 (3개 묶음) | 120G | 감빗 "회복 물약"의 탄약 |
| 미끼 | 80G | 배치형 1개 + 투척용 1개 |
| 연막탄 | 100G | 도주 저지 보험 |
| 폭탄 (2개 묶음) | 150G | 광역 딜 보충 |

**정보**

| 품목 | 가격 | 비고 |
|---|---|---|
| 정찰 1회 | 200G | 미공개 정보 1개 공개 (약점 → 패턴 → 숨은 기믹 순) |
| 정밀 정찰 (랭크 3 해금) | 350G | 정보 1개 + 함정 배치 화면 동선 선명 표시 |

### 7.3 의뢰 보상표

| 몬스터 | 준비 골드 | 기본 보상 (명예 pt) | 풀세팅 총액 |
|---|---|---|---|
| 1. 멧돼지 왕 | 800G | 100pt | ~600G (일부러 여유 — 튜토리얼) |
| 2. 질풍 사슴 | 1,000G | 180pt | ~1,400G |
| 3. 화염 곰 | 1,100G | 250pt | ~1,500G |
| 4. 그림자 살쾡이 | 1,200G | 320pt | ~1,700G |
| 5. 서리 독수리 | 1,400G | 500pt | ~2,000G |

2번부터 예산 = 풀세팅의 약 70% 고정 (원칙 3의 수치화).

**예산 검증 — 화염 곰 (1,100G):**

- *버티기 빌드*: 화염 내성 150 + 방어구 Lv1 200 + 물약 120 + 무기 Lv1 200 + 정찰 2회 400
  (격노 해제 기믹 필수 공개) = **1,070G** — 함정을 하나도 못 삼
- *폭딜 빌드*: 무기 Lv2 400 + 전격 트랩×2 300 + 낙석 250 + 미끼 80 = **1,030G** —
  정찰·내성 포기. 격노 전에 못 끝내면 맨몸으로 3배 공격을 받는 도박
- 도감이 쌓이면(정찰 불필요) 버티기 빌드에 400G 여유 발생 → **재도전이 경제적으로 쉬워짐**

### 7.4 보상 배율 정산식

```
최종 명예 pt = 기본 보상 × (100% + 보너스 합산)
```

| 보너스 | 값 | 상한 |
|---|---|---|
| 남은 시간 10초당 | +5% | +60% (2분 컷) |
| 부위 파괴 1곳당 | +10% | +30% (3부위) |
| 긴급 카드 미사용 1장당 | +10% | +30% |
| 노 기절 클리어 | +20% | +20% |

- 이론상 최대 240%, 현실적인 잘한 판은 150~180%
- **실패 시**: 명예 pt 0. 단 도감 기록 유지 + **위로금 30pt** (메타 성장 완전 정지 방지)

### 7.5 메타 경제 (명예 포인트 사용처)

| 해금 | 비용 | 내용 |
|---|---|---|
| 랭크 2 승급 | 400pt | 감빗 슬롯 +1(→4), 페이즈 조건군, 그물 발사기 입고, 의뢰 4·5 개방 |
| 랭크 3 승급 | 1,000pt | 감빗 슬롯 +2(→6), 위치 조건군, 정밀 정찰 입고 |
| 긴급 카드 신규 1종 | 150~300pt | 카드 풀 확장 |
| 시작 골드 +100G (2회 한정) | 300pt/회 | 유일한 직접 전투력 구매 — 2회 제한으로 인플레 차단 |

**페이스 검증**: 의뢰 1~3을 평균 배율 130%로 깨면 약 655pt → 랭크 2 승급 + 카드 1종.
의뢰 4~5와 재도전 파밍으로 랭크 3까지 약 8~12판 = **첫 사이클 3~4시간**.

### 7.6 경제 붕괴 방지 장치

- 명예 pt로 골드를 못 사게 (시작 골드 +100 2회 제외) — 메타 파밍으로 판 내 긴장감이 무너지는 것 차단
- **도감이 최고의 화폐**: 정찰비 200~400G 절약이 재도전 보너스 역할 —
  별도 버프 없이 "아는 만큼 싸진다"가 성립

## 9. 프로토타입 검증으로 확정된 규칙 개정 (v0.2)

웹 프로토타입의 헤드리스 시뮬레이션(시나리오별 5~20회 반복)으로 검증하며 확정한 개정 사항.
§2~§5의 원칙과 충돌하는 부분은 이 절이 우선한다.

### 9.1 헌터 지능: 함정 유인 (신규 감빗 행동)

함정은 "우연히 밟는 장치"가 아니라 헌터가 능동적으로 쓰는 무기다.
감빗 행동 "함정 유인"은 몬스터 유형에 따라 다르게 동작한다:

- **공격형(멧돼지)** — 미끼 역할: 함정 너머 1.6유닛 지점에 서서 돌진이 함정을 지나가게 유도
- **회피형(사슴)** — 몰이: 사슴 기준 함정 반대편 2.8유닛에 붙어 도망 방향을 함정 쪽으로 강제
- 유인 지능은 **탈출구 길목(게이트 반경 2유닛)의 함정을 아낀다** — 교전 유인에 소모하면
  도주 저지 수단이 사라지기 때문 (시뮬레이션에서 실제 발생한 실패 패턴)
- 기본 감빗도 4슬롯으로 확장: `함정 걸림→강공격 / 빈틈→평타 / 내 HP<50%→물약 / 항상→함정 유인`

### 9.2 유효타 창 (몬스터 장갑)

"탱킹하며 맞딜"을 막고 유인→폭딜 리듬을 만들기 위해:

- **멧돼지 왕**: 평상시 받는 피해 30%. **경직(돌진 후)·함정 걸림 때만 100%**
- 신규 감빗 조건 **"빈틈(경직·탈진)"**: 멧돼지 경직 또는 사슴 탈진 중 참

### 9.3 함정 관련 수치 개정

| 규칙 | 기존 | 개정 | 사유 |
|---|---|---|---|
| 함정 걸림 피해 배율 | 2.0배 | **1.5배** | 속박 4초 풀버스트 원킬 방지 |
| 함정 해제 후 | — | **6초 함정 내성** | 연쇄 스턴 방지. 함정을 한 곳에 몰면 낭비 |
| 함정 내성 예외 | — | **도주 중엔 내성 무시** | 저지 불가 창이 생기는 조합 버그 방지 |
| 도주 저지 성공 시 | 게이지 리셋만 | **+25초 탈진 (이속 -28%)** | 저지가 실제 처치 기회가 되도록 |

### 9.4 몬스터 개정

- **멧돼지 왕**: HP 300 → **450** (장갑 도입에 맞춰 재조정)
- **질풍 사슴**: HP 250 → **320**. **함정 회피 지능** 추가 — 진행 방향 앞의 함정을 보면
  꺾어서 피한다. 단 **탈진 중·도주 중에는 회피 불가** (몰이의 밀당과 저지의 보상)

### 9.5 검증 결과 (v0.2 기준)

| 시나리오 | 결과 |
|---|---|
| 멧돼지 / 맨몸 | 승률 100%, 평균 71초, 매판 기절 — "준비 없으면 고생" |
| 멧돼지 / 함정 2개 + 유인 감빗 | 승률 100%, 평균 15초, 무피해 — 준비의 보상 |
| 사슴 / 함정 없음 | 승률 0% — 열쇠 없이는 불가 |
| 사슴 / 양쪽 길목 + 중앙 함정 | 승률 90%, 평균 82초 — 의도한 표준 공략 |
| 사슴 / 한쪽 길목만 (도박) | 승률 70%, 평균 62초 — 리스크-리턴 |

Unity 프로토타입(§8)은 v0.1 규칙 기준이며, 웹 프로토타입(`WebPrototype/`)이
현행 튜닝 기준이다. 수치가 안정되면 Unity에 일괄 반영한다.

### 9.6 AI·이동 개정 (v0.3) — "몹이 가장자리만 돈다" 문제 해결

실플레이 피드백("몬스터가 테두리만 돌아 함정을 안 밟는다")으로 발견된 이동 AI 결함과 개선:

**몬스터 이동 지능**
- 배회 경로: 고정 사각 순환(테두리 루프) → **아레나 전역 무작위 지점** — 중앙 함정 지대를 자연히 통과
- **벽 반발 조향**: 벽 1.5유닛 이내에서 중앙 방향 힘을 섞는다 — 벽을 타고 도는 이동 제거
- 사슴은 압박이 없을 때 **안쪽으로 이동하며 쉼** (벽에 붙어 정지하지 않음)
- **몬스터 시작 위치 4곳 중 무작위** — 스폰 옆 함정 캠핑(치즈) 방지

**헌터 유인 지능 강화**
- 멧돼지 조준 중 돌진선이 이미 함정을 지나면 **자리를 지키며 버틴다** (이동으로 조준선을 흐트리지 않음)

**신규 규칙: 몸부림**
- 함정에 걸린 몬스터는 **맞을 때마다 걸림 시간 -0.4초** — 폭딜일수록 창이 짧아진다.
  속박 1개 + 폭딜 무기 원킬 조합을 자기조절식으로 억제 (별도 수치 너프 불필요)
- 도주 결심은 **함정 내성 창(6초) 동안 지연** — "막 풀려난 사슴은 방향감각을 잃는다".
  교전 중 길목 덫이 소모된 직후 즉시 도주로 빠져나가는 미저지 창 제거

**v0.3 검증 결과**

| 시나리오 | v0.2 | v0.3 |
|---|---|---|
| 멧돼지 / 맨몸 | 100%, 71초 | 100%, 69초 |
| 사슴 / 양쪽 길목 + 중앙 | 90%, 82초 | **95%, 48초** |
| 사슴 / 한쪽 길목 올인(무기Lv2) | 70% | **60% (진짜 도박)** |
| 사슴 / 함정 없음 | 0% | 0% |
| 중앙 함정 발동률 (사슴전) | 거의 0 | 평균 1.0개/판 |

### 9.7 유저 피드백 반영 — UX 대개편 (v0.4)

실플레이 피드백("맵이 작다 / 몬스터가 도망만 다닌다 / 대각선 시점이 불편 / UI가 불친절")을
유저 여정 시뮬레이션으로 감사한 뒤 반영한 대개편.

**시점·화면**
- 아이소메트릭(등각) → **탑다운 복귀**. 디아블로 아트(석판·핏자국·해골·성벽·횃불·고딕 UI·체력 오브)는 전부 유지.
  등각은 화면의 ~40%를 여백으로 낭비 + 클릭 정밀도·거리감이 나빴다.
- 캔버스 **풀폭 확대**(840×520 → 1100×620), 로그를 우측 레일에서 **캔버스 오버레이**로 이동.
- 아레나 월드 확대: ARENA 7.5×4.5 → **9×5.2** (체감 맵 크기 대폭 증가).

**"도망만 다닌다" 해소 — 전투 밀도**
- 헌터 평타를 **원거리 석궁 볼트**로 변경(피해 7, 유도). 도망가는 상대에게도 딜이 들어가
  "쫓는 시간"이 진행으로 읽힌다. 멧돼지 장갑(30%)은 유지되어 함정 필요성은 그대로.
- 질풍 사슴에 **방심(off-guard)**: 주기적으로 멈춰 숨을 고름(유효타 창 + 접근 기회).
- 질풍 사슴에 **뿔치기 반격 돌진**: 가끔 헌터에게 달려들어 들이받고 재도주.
  → "도주 → 방심 → 접근 → 반격 → 재도주" 리듬이 생겨 관전 밀도 확보.

**UI 친절도 (신규 유저 여정)**
- **추천 세트 원클릭 구매** 버튼(몬스터별 권장 구성). 선택 마비 해소.
- 상점 **환불** 버튼(구매 되돌리기). 배치 화면 **함정 회수**(설치된 함정 클릭 → 재고 복귀).
- **도주 카운트다운 게이지**(HUD): "도주까지 M:SS · 또는 HP 40%" / 도주 중·탈진 상태 표시.
- **스폰 후보 4곳(★) 표시** + 몬스터 머리 위 **상태 아이콘**(🕸 함정 / 💤 방심 / 🏃 도주 / ❗ 조준).
- 정산 실패 시 **원인별 구체 팁**(도주 실패 → 길목 속박 덫 / 시간초과 → 유효타 창 만들기).
- 감빗 안내 문구("기본 전술이 이미 세팅됨 — 그대로 시작해도 됨").

**밸런스 재조정 (시뮬레이션 검증)**
- 멧돼지 HP 380 / 돌진 피해 18 (튜토리얼 완화), 질풍 사슴 HP 520 (도주 국면 유지).
- 검증: 멧돼지 맨몸 100% 승(114초·고전) vs 추천세트 100%(20초·무피해);
  사슴 맨몸 0%(전부 도주) vs 추천세트 ~85%(평균 32초, 도주 저지 드라마 유지).

### 9.8 손맛·재도전 (v0.5)

유저 여정 시뮬레이션에서 "관전이 밋밋하다 / 실패 후 다시 준비하기 번거롭다"로 지목된 두 지점을 보강.

**사운드 + 타격 연출 (juice)**
- **WebAudio 절차 생성** 효과음 — 외부 파일 0개. 오실레이터 톤(글라이드) + 필터 노이즈 버스트로
  석궁 발사 / 명중 / 강공격 / 속박·마비 함정 / 도주 저지 / 피격 / 기절 / 물약 / 카드 / 처치·승패를 각각 합성.
  첫 사용자 제스처에서 AudioContext를 깨우고(자동재생 정책), HUD의 🔊 버튼으로 토글(localStorage 저장).
- **화면 흔들림·플래시**: 명중(약)·함정 발동(황색)·마비(청색)·도주 저지(녹색)·피격(적색)·기절(강)·처치(백색).
  `hunt.shake`(실시간 감쇠)를 캔버스 translate에, `hunt.flash`를 전체 오버레이로 적용 — 판정 로직과 분리되어 밸런스 불변.

**같은 세팅으로 재도전**
- 사냥 시작 시 준비 상태(구매·감빗·함정 배치 좌표)를 스냅샷.
- 정산 화면에 **[↻ 같은 세팅으로 재도전]** 추가 — 준비 골드 소모 없이 동일 구성으로 즉시 재사냥
  (물약·카드·함정 재장전). "다음 의뢰"는 새 준비로 초기화. 실패 후 미세 조정 반복 학습을 매끄럽게.
