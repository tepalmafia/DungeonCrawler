# 밸런스 프레임워크

모든 수치 조정의 기준. "감"이 아니라 이 문서의 목표치에 맞춘다.

## 1. 파워 커브 기준

### 플레이어 기대 성장 (보스 도달 시점) — XP 커브: 시작 36, 배율 ×1.24

| 층 보스 | 기대 레벨 | 층 보스 | 기대 레벨 |
|---|---|---|---|
| 1층 | 4~5 | 6층 | 16~17 |
| 2층 | 7~8 | 7층 | 17~18 |
| 3층 | 10~11 | 8층 | 19~20 |
| 4층 | 12~13 | 9층 | 20~21 |
| 5층 | 14~15 | 10층 | 21~22 |

원칙: 심층(6~10층)에서도 층당 1~2레벨은 오른다 — 새 몬스터가 나오는 구간에서 성장이 멈추지 않게.

### 특성 중첩 상한

곱연산(지수 성장) 특성은 낮게, 덧셈 특성은 넉넉하게. 상한 도달 시 카드 풀에서 제외.

| 특성 | 상한 | 이유 |
|---|---|---|
| 행운 | 2 | ×2 곱 — 3중첩이면 하트 48%로 경제 붕괴. 유물(클로버)과 합산한 **총 배율도 ×3 상한** |
| 바람걸음 / 회전 가속 / 신속 장전 / 콤보 / 탐욕 | 3 | 곱연산·시스템 우회 |
| 신속 / 질풍 / 장검술 | 4 | 곱연산 |
| 파괴자 | 5 / 급소 간파·강골 | 6 / 힘 단련 | 8 | 덧셈 — 쌓는 재미 유지 |

### 적 스케일

- 적 HP 배율: 1~5층 ×(1 + 0.3×(층−1)) — 5층에서 ×2.2. **6층+ 기울기 +0.4/층** (10층 ×4.2) — 계측에서 5층부터 받은 피해가 거의 0 (성장이 +0.3 기울기를 추월)
- **실효 HP 톱니 보정 (2026-07)**: 층 풀 교체로 실효 HP가 5~6층 -40%·9층 -50% 급락하던 문제 — 저HP 풀에 고HP 몬스터 투입: 5층 +골렘(10), 6층 +덩치(13), 9층 +골렘. 재계측: 6층 실효 HP 9.7~12.9 → 13.6~18.3 (4층과 자연 연결), 9·10층 소폭 상승
- 적 속도: 층당 +2% (10층에서 +18%)
- 적 수: 방 깊이 + 층 보정, 최대 16 — **1~2층은 −2** (곡선 뒤집기 R1: 사망 90%가 1~2층에 몰리는 역전 보정)
- **심층 압박 (R1)**: 7층+ 정예·우두머리 접촉 피해 2 — 후반 순항 방지
- **위협 세트 (R2)**: 전투방 45%는 손제작 조합 10종(방패벽+저격수 / 힐러+덩치 / 자폭+돌진 등)이 무리의 뼈대 — 물량이 아니라 우선순위 퍼즐
- **지름길 (R3)**: 3·6층 보스 후 갈림길 — 한 층 스킵, 도착 층은 정예 2배(상한 50%)·우두머리 25%+2연속 허용·정예가 파편 4 드랍
- 정예 확률: 3% + 4%/층
- 하트 드랍: 기본 4.5% × max(0.4, 1 − 0.04×(층−1) − **7층+ 추가 −0.12**) × 행운 배율(**총 상한 ×3**) — 심층일수록 회복이 귀하다 (열기 4+ 절반, 보스전 절반)
  - **풀피 감쇠 ×0.35**: 못 먹는 하트가 바닥에 쌓이는 낭비 방지 (다치면 정상 확률로 복귀)
  - **방당 소프트캡**: 한 방에서 3개째부터 확률 ×0.25 — 물량방 하트 인플레 차단
- 열기: HP+25% / 수+2 / 속도+15% / 회복↓ / 보스+50%·시작HP−1 (누적)
- 보물·모닥불 문: **층당 각 1회만** 등장 (모닥불은 방 4 이후) — 유물·회복 공급 상한
- **모닥불 방 양자택일**: 휴식(HP +2, 열기 4+ 시 +1) vs 담금질(이번 층 동안 공격력 +1, 층 이동 시 만료) — 하트 수급이 충분해진 뒤에도 모닥불 방이 죽은 선택지가 되지 않게. 하나를 고르면 다른 쪽은 사라진다
- 무한 모드(11층+): 테마·풀·각성 보스 순환, 보스 HP 층당 +15%, 정예 확률 상한 40%, 적 속도 상한 +30%, 파편은 10층 정산 초과분만 추가 지급
- 심층(6~10층) 전용 몬스터: 폭탄벌레(자폭·보상 없음) / 가시덩굴(고정 포탑 산탄) / 처형자(직사각 텔레그래프 도끼) / 마그마 슬라임(사망 시 분열) / 공허의 눈(순간이동 + 추적탄)
- **우두머리(중간보스)**: **3층+ 층당 최소 1회 보장** (방마다 14%, 보스 전 4방까지 미등장 시 확정 난입). 2층은 14% 확률만 — 확정 등장은 초반 약체 빌드에 과했다 (계측: 검사 2층 사망 1→11회 급증해 롤백). 시뮬 300층: 2층 0.56회 / 3~5층 1.0회(0회 확률 0.7%↓) / 심층 1.3회. 심층(6층+)은 두 번째 우두머리 12%. HP×7·크기 1.5배·XP×8, 어픽스 1개(6층+ 2개): 신속/강철(장갑2)/재생(2/s)/유령(주기 비물질)/소환(5초마다 2기)/폭발(사망 시 r90)/분열(사망 시 2기). 처치 보상: 하트 확정 + 파편 6+층×2 즉시

### 보스 TTK(처치 소요 시간) 목표

| 보스 | 정답 트리 보유 시 | 미보유 시 |
|---|---|---|
| 1층 카론 | 30~40초 | 40~50초 (트리 불필요 — 입문) |
| 2층 믹서스 | 40~50초 | **재생으로 사실상 교착** |
| 3층 바르곤 | 45~55초 | **장갑으로 2배 이상 소요** |
| 4층 이그니스 | 50~60초 | **가속 누적으로 후반 붕괴** |
| 5층 눅스 | 60~80초 | **장막 실패 시 회복 반복** |

원칙: **1층은 조작으로, 2층부터는 빌드로.** 층이 오를수록 "정답 트리 없이는 낮은 확률, 있으면 높은 확률"이 되도록 기믹 체크를 설계한다.

## 2. 보스 기믹 체크 (연구 요소)

각 보스는 스탯만으로는 뚫기 어려운 메커니즘을 갖고, 특정 특성 트리가 그 해답이 된다.
보스 체력바 아래에 기믹 이름만 표시한다 — **해법은 플레이어가 연구**한다. (도감에 힌트)

| 보스 | 기믹 | 작동 방식 | 정답 트리 (스포일러) |
|---|---|---|---|
| 2층 믹서스 | **포자 갑피** | 부하(소환수)가 살아있는 동안 초당 6 재생. **부하 처치 시 재생 5초 정지** (컨트롤 해법) | 광역 처치: 🔥화상 폭발 / 독 역병 / ⚡연쇄 번개 |
| 3층 바르곤 | **중장갑** | 한 번의 직접 타격 피해를 최대 2로 경감 | 경감 무시 지속 피해: 🔥점화 / 독날, 부식(+1도 유효) |
| 4층 이그니스 | **백열** | 16초마다 패턴 가속 (최대 4중첩), 불길 누적 | 속전: 크리 트리(급소·파괴자·발화점) 또는 유지력: 🩸흡혈/회복력/🛡️보호막 |
| 5층 눅스 | **어둠 장막** | HP 70%·35%에서 무적화 + 영혼 구슬 3개 소환. 8초 내 전부 파괴 못 하면 15% 회복, 성공하면 긴 그로기 | 다중 표적 대응: ⚡잔전류+과충전(대시 기동) / 연쇄 번개 / 원거리 직업 |

- 직접 타격(타격감 연출이 있는 공격)만 장갑 경감 대상. 화상·중독 틱, 폭발·연쇄·장판은 **경감을 무시**한다 — 이것이 연구 포인트.
- 보스 HP (2026-07 스킬 상향 반영 재조정): 카론 95 / 믹서스 220 / 바르곤 190 / 이그니스 430 / 눅스 560
- 6~10층(심층) 각성 보스: 원혼 카론 550 / 역병왕 680 / 절망의 바르곤 650 / 겁화의 이그니스 850 / 진 눅스 1050·장막 3회

### 긴장감 장치 (2026-07 — "보스가 너무 쉽다" 피드백)

- **버스트 상한 (2026-07)**: 보스 대상 타격당 최대 피해 = 최대 HP의 1.5% (최소 2) — "골렘 외 보스가 강빌드에 녹아서 기믹을 못 본다" 피드백. 기준점은 유일하게 보스답다는 평가를 받은 골렘(중장갑, 강빌드 TTK 29s). 약한 빌드에는 상한이 안 걸려 저점 난이도 불변. 강빌드(공8·크리60%·최대 업타임) 실측 TTK: 카론 15.6s / 믹서스 36.4s / 바르곤 29.2s / 이그니스 24.2s / 눅스 51.5s / 진눅스 56.7s — 전 보스 페이즈2·장막 발동 확인
- **소프트 인레이지**: 모든 보스는 전투 45초마다 패턴 대기 ×0.85 (최대 3중첩) — 오래 끌수록 위험하다
- **2페이즈 접촉 피해 2** — 반피 이후엔 몸도 흉기
- **장막 실패 회복 15%→25%**, 영혼 구슬 HP 2→3×배율 — 광역기로 즉시 청소되지 않게
- **보스전 중 부하 하트 드랍 절반** — 부하가 회복 셔틀이 되지 않게

## 2.5 직업 밸런스 원칙

- 원거리(궁수·마도사)는 낮은 HP(4/3)와 느린 공격 쿨로, 근접(검사)은 높은 HP(6)·긴 피격 무적(1.2s vs 0.9s)·넓은 근접 범위(77/94)로 보상한다.
- 검사가 체감상 어렵다는 피드백 반영 이력 (2026-07, 3차):
  ① HP 5→6, 피격 무적 0.9→1.2s, 근접 사거리 +10% ② 전투 본능(마무리 적중 시 HP 1 회복)
  ③ 철벽(11초마다 보호막 — 피해 1회 무효) + 전투 본능 쿨 6→4s
  텔레메트리(QA 봇 10게임분 사망): 전사 19~20회 → 7~8회 (궁수 1, 마도사 0~13) — 근접 리스크 감안 격차 수용
- 물량감: 방당 적 수 상향 (기본 3 + 깊이×0.9 + 층×0.9, 상한 16). 보정: 마리당 XP ×0.75, 하트 6%→4.5% — 총 공급은 이전과 동일 수준
- **연출 부하 가드 (물량전 타격 렉 방지, 2026-07)**: ① 효과음 스로틀 — 같은 효과음(타격/크리/처치/발사/폭발)은 45~60ms 창에 2회까지만 재생 (광역기 몰살 시 오디오 노드 폭주 차단) ② 파티클 전역 상한 — 입자 500 / 데미지 숫자 28 / 링 24 / 스타 12, 상한 근접 시 밀도 자동 축소 ③ 히트스톱 완화 — "탁탁 멈추는 게 렉 같고 눈 아프다" 피드백: 일반 타격은 멈춤 없음(흔들림+파티클+사운드가 손맛 담당), 크리 0.055s·잡몹 처치 0.05s는 0.35초 창에 1회만, 정예·우두머리·보스 처치는 항상 0.09s. 크리 화면 섬광도 같은 리듬으로만 + 강도 0.12→0.08 (고크리 빌드 연속 번쩍임 방지). 계측: 24마리 물량방 12초 전투에서 33ms 초과 프레임 2→0, 최대 프레임 33.4→16.8ms
- 잡몹도 전부 예고 동작 있는 공격 패턴 보유: 슬라임 도약(웅크림 0.45s), 박쥐 급강하(정지 0.4s)
- **무한 대시 차단 (2026-07)**: 대시 충전 하한 0.6초 — 어떤 조합(제단 바람 3강 × 바람걸음 3중첩 × 깃털 = 0.48초)이라도 하한을 지킨다. 폭주 기관(레전더리)은 '쿨다운 소멸' → **0.45초 고정**으로 재설계 (하한보다 빠른 유일한 수단, 무한은 불가). 과충전(감전 처치 시 완전 충전)은 **2초에 1회** — 물량방 연쇄 처치로 상시 충전되는 구멍 봉쇄. 근거: 대시 무적 0.22초가 상시 유지되면 회피가 아니라 면역이 된다

### 직업 스킬 (2026-07 상향 — "기본 공격만 쓰게 된다" 피드백)

| 스킬 | 쿨 | 위력 | 고유 유틸리티 |

- 준비 완료 시 차임 + "준비!" 표시 (잊고 안 쓰는 문제 대응)

## 2.6 방 템플릿 사망률 (2026-07 계측 — 봇 4런, 방문당 사망)

- 측정법: 봇 4런(검사×2·궁수·마도사, 무한 부활, ff8) — 전투방 방문마다 템플릿 기록, 사망·체류 시간 귀속. 198방 표본
- 1차 계측 중앙값 ~0.2 사망/방문. 이상치 3종 조정:
  | 템플릿 | 1차 | 원인 | 조정 |
  |---|---|---|---|
  | 외곽 링 | **1.86** (체류 76s) | 출입구 2개 밀실 — 몰이당하면 탈출 불가 | 상하 개구부 추가 (4방향 출입) |
  | 균사 미로 | **1.13** | 돌출벽 + 독 안개 = 구석 몰이 포켓 | 돌출벽 제거, 위험을 보이는 독 웅덩이로 |
  | 용암 강 | 0.75 | 1칸 다리에서 넉백 낙사 | 다리 2칸 폭 |
- 조정 후 재계측(3차): 외곽 링 1.86→**0**, 균사 미로 1.13→**0.11**, 용암 강 0.75→**0.2** — 전 템플릿 0~0.33 균질 밴드 (상위는 용암 계열 0.2~0.33 = 테마 의도 위험). 2차에서 튄 쌍둥이 방책 1.14는 3차 0 — 런 분산으로 판정
- 부수 발견: 제자리 부활이 용암 호수 한복판에서 998연속 사망 루프를 만들었다 → **테스트 부활은 방 입구로 이동** (실플레이 영향 없음 — 진짜 사망은 런 종료, 불사조 부활은 기존대로 제자리)

## 3. 트리 연구 지원 — 카드 등장 가중치

특성 카드는 **이미 보유한 태그와 같은 태그가 더 자주 등장**한다 (가중치 1 + 0.7×보유 수, 스탯 태그는 0.2배 감쇠).
