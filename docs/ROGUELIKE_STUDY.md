# 로그라이크 명작 분석 — 무엇이 호평을 만들었고, 우리는 무엇을 가져올 것인가

> 대상: 던전 크롤러: 심연의 탑. 기준 문서: GAME_DESIGN.md, BALANCE.md.
> 분석 대상은 평단·판매 양쪽에서 대성공한 로그라이크/로그라이트 8종.

## 1. 게임별 핵심 성공 요인

| 게임 | 무엇으로 성공했나 | 한 줄 교훈 |
|---|---|---|
| **Hades** | 죽음마다 스토리가 전진 — 패배가 손실이 아니라 서사의 페이지. 신들의 축복(특성) 조합이 매 런 다른 무기 궁합을 만든다 | **죽음에 의미를 부여하라** |
| **Slay the Spire** | 카드 한 장 한 장이 빌드 결정 — "이번 런은 무슨 아키타입인가"가 3~4선택 안에 잡히고, 남은 런은 그 가설의 검증 | **선택의 밀도 = 재미의 밀도** |
| **The Binding of Isaac** | 아이템이 능력치가 아니라 **공격 방식 자체를 바꾼다** — 눈물이 레이저가 되고 몸이 변한다. 조합 발견이 곧 콘텐츠 | **파워업은 눈에 보여야 한다** |
| **Dead Cells** | 60fps 손맛 + 무기마다 완전히 다른 플레이 리듬. "한 판만 더"를 만드는 즉각 재시작 | **조작감이 로그라이크의 바닥짐** |
| **Enter the Gungeon** | 탄막 회피라는 명확한 실력 축 — 죽으면 언제나 "내가 피할 수 있었다" | **실패는 항상 플레이어 귀속** |
| **Risk of Rain 2** | 시간이 곧 난이도 — 꾸물거리면 세계가 강해진다. 아이템 무한 중첩의 폭주 쾌감 | **시간 압박이 결정을 강제한다** |
| **Vampire Survivors** | 물량 대학살 파워 판타지 + 무기 진화(재료 조합 → 상위 무기)의 수집욕 | **스킬/무기의 '진화' 순간을 만들라** |
| **Balatro** | 숫자가 지수적으로 불어나는 순간의 도파민 — 빌드가 완성되면 게임의 룰 자체를 부수는 느낌 | **빌드 완성의 '폭발 순간'을 허용하라** |

## 2. 공통 성공 공식 5가지 — 우리 게임 충족도

| 공식 | 내용 | 우리 상태 |
|---|---|---|
| ① 런 서사 | 매 런이 "다른 이야기"가 되는 빌드 분기 | **◎** 특성 43종(태그 시너지)·유물 20종·전설 특성·기연 도박 |
| ② 죽음의 가치 | 죽어도 남는 것 — 메타 성장, 지식, 서사 | **○** 파편·제단·도감은 있음. 죽음 자체의 서사/정보는 없음 |
| ③ 결정 밀도 | 몇 초마다 유의미한 선택이 오는가 | **◎** 문 선택·카드 3택·기연·모닥불 양자택일·저주 상자·피의 제단 |
| ④ 실패 귀속 | "게임이 죽인 게 아니라 내가 죽었다" | **◎** 전 몬스터 텔레그래프 공격·회피 가능 설계 |
| ⑤ 파워 판타지 | 빌드가 완성됐을 때의 폭주 체감 | **△** 잡몹방에선 폭발 허용(물량 학살), 그러나 **체감 연출이 약하다** |

결론: 구조(①③④)는 이미 명작 문법을 따른다. 부족한 것은 **②죽음의 가치**와 **⑤파워의 체감**, 즉 "숫자는 강해졌는데 눈과 손이 그걸 덜 느낀다"는 축이다.

## 3. 적용 제안 — 우선순위 순 (P1~P4 구현 완료 2026-07, P5 검토 대기)

### P1. 스킬 진화 (Vampire Survivors / Isaac)
특성 중첩이 상한에 닿으면 직업 스킬이 **진화**한다 — 숫자 증가가 아니라 **형태 변화**.
- 예: 회전 베기 상한(3) 도달 → **회오리 베기**(이동하며 시전); 화살비 상한 → **유성우**(낙하 범위 2배·화상); 메테오 → **쌍둥이 메테오**.
- 이유: 우리의 상한 시스템(밸런스 장치)을 **보상 순간**으로 뒤집는다. 상한이 "끝"이 아니라 "완성"이 된다. Isaac의 교훈(눈에 보이는 파워업)을 가장 적은 신규 시스템으로 달성.

### P2. 사망 리포트 — 죽음에 정보 가치 부여 (Hades 경량판)
스토리가 없는 우리 게임에서 Hades식 서사는 과하다. 대신 죽음을 **지식**으로 바꾼다.
- 정산 화면에 : 사인(무엇에게·어느 층·무슨 공격), 이번 런 빌드 요약, "지난 런보다 나아진 점"(도달 층/처치/파편) 한 줄.
- 이미 QA 봇용 층별 사망 집계 인프라가 있다 — 플레이어용으로 노출만 하면 된다.

### P3. 유물의 시각 피드백 (Isaac)
유물 획득이 스탯 변화로만 남는다. 최소 비용 규칙: **유물 1개당 눈에 보이는 흔적 1개**.
- 광전사 투구 → 플레이어 주변 붉은 아우라, 사슬 갑옷 → 피격 시 금속 스파크, 클로버 → 하트 드랍 시 초록 반짝임.
- 전부는 어렵더라도 레어 이상 8종부터. "내 캐릭터가 런마다 다르게 보인다"가 목표.

### P4. 일일 시드 도전 (Spelunky / StS 데일리)
이미 `?seed=` 인프라가 있다. 날짜 기반 고정 시드 + 고정 직업 + 결과 기록(로컬)만 붙이면
"오늘의 탑"이 된다. 같은 던전을 모두가 도전한다는 소셜 훅 — 구현 비용 대비 재방문 유인이 가장 크다.

### P5. (검토) 시간 압박 요소 (Risk of Rain 2)
층 체류 시간이 길수록 정예 확률이 소폭 오르는 **"심연의 시선"** 게이지. 소극 플레이(방마다 풀정비)를
자연 억제하고 결정에 긴장을 더한다. 단, 탐험 재미와 충돌할 수 있어 **무한 모드 한정 도입** 후 반응을 본다.

## 4. 맵 디자인 분석 — "단조롭다"의 원인과 명작의 해법 (2026-07 추가)

### 명작들의 맵 생성 방식

| 게임 | 방식 | 핵심 |
|---|---|---|
| **Spelunky** | 절차 생성의 교과서 — 4×4 방 **템플릿** 조합 + 반드시 통과 가능한 '솔루션 경로' 보장 + 파괴 가능 지형 | 랜덤은 조각의 '배치'에만, 조각 자체는 손제작 |
| **The Binding of Isaac** | 손제작 방 수백 개를 풀에서 뽑아 층 그래프에 배치 (비밀방·보물방의 위상 규칙) | **방 지형 = 퍼즐.** 바위·구덩이 배치가 그 방의 공략 |
| **Enter the Gungeon** | 손제작 방 풀 + 랜덤 연결. 기둥·뒤집는 테이블 = 탄막 **엄폐** | 지형이 회피 메커니즘과 상호작용 |
| **Hades** | 방 자체는 고정 아레나 소수, 랜덤은 '순서와 보상'에 집중 | 방 하나하나가 조탁된 전투 무대 |
| **Dead Cells** | 손제작 '조각(prefab chunk)'을 절차적으로 이어붙임 + 수직성 + 지름길 분기 | 조각 단위 손제작 + 연결만 랜덤 |
| **Risk of Rain 2** | 대형 고정 맵 소수 + 오브젝트/포탈만 랜덤 | 맵 암기가 실력이 되는 축 |

**공통 원칙 4가지**: ① 완전 랜덤이 아니라 **손제작 조각의 랜덤 조합** ② 지형이 전투와 **상호작용**한다 (엄폐·좁은 목·위험지대·파괴물) ③ 탐색에 **보상**이 있다 (비밀·분기) ④ 방의 **실루엣**이 한눈에 다르다.

### 우리 게임 진단

현재: 직사각 아레나 + 무작위 벽 블록 점뿌리기 + 층 환경(용암/안개/얼음) + 문 분기.
단조로움의 원인 = 원칙 위반 지점:
- **①위반**: 벽이 '조각'이 아니라 무작위 점 — 지형이 의미 있는 형태로 안 읽힌다
- **②약함**: 벽이 엄폐/동선 설계가 아니라 장애물 노이즈다 (봇 LoS 문제를 만들던 그 벽)
- **③없음**: 방 안 탐색 보상이 없다 — 방 = 전투 후 통과
- **④위반**: 모든 방이 같은 직사각 실루엣

### 적용 로드맵 (우선순위)

- **M1. 손제작 방 템플릿 풀** — 기둥 홀 / 중앙 용암 호수 / 외곽 링 / 좁은 목 협곡 / 십자 통로 / 대각 벽 / 소형 미로 등 12~20개를 문자열 타일맵으로 정의하고 랜덤 추첨 (+ 좌우 반전으로 2배). 현 코드가 이미 문자열 픽셀맵 문화라 구조 변경 없이 들어간다. **비용 대비 효과 최대.**
- **M2. 지형-전투 상호작용** — 원거리 몹 사선을 끊는 기둥, 물량을 조이는 좁은 목, 파괴 가능 항아리/벽(파편 소량) — 템플릿에 문자 하나 추가로 표현
- **M3. 비밀실** — 균열 벽(미세하게 다른 타일) 뒤 소형 방: 파편/하트/가끔 유물. 탐색 보상 축 신설
- **M4. 실루엣 변형** — L자/십자/타원 마스크 (M1 템플릿에 흡수 가능)

## 5. 하지 말 것 (분석이 주는 반면교사)

- **메타 성장으로 난이도를 사지 않기** — 제단 업그레이드가 강해지는 유일한 길이 되면 런 내 선택이 값싸진다 (우리 제단은 7종·소폭 유지가 맞다).
- **무기/직업 남발 금지** — Dead Cells의 무기 다양성은 각각이 완전히 다른 리듬이라 성립한다. 어설픈 4번째 직업보다 스킬 진화(P1)가 낫다.
- **RNG에 실패를 떠넘기지 않기** — 기연은 도박이지만 "받을지 말지"는 항상 플레이어 선택이다. 이 원칙(수락 전 이탈 가능)을 앞으로도 지킨다.
