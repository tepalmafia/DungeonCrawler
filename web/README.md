# 던전 크롤러: 심연의 탑 (웹 버전)

외부 프로그램·라이브러리 없이 순수 HTML5 + Canvas + Web Audio로 만든 로그라이크 던전 크롤러 액션 RPG.
기획안: [`docs/GAME_DESIGN.md`](../docs/GAME_DESIGN.md)

## 실행 방법

`web/index.html`을 브라우저에서 열면 바로 실행됩니다. 또는 로컬 서버로:

```bash
cd web
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 조작

| 키 | 동작 |
|---|---|
| WASD / 방향키 | 이동 |
| 마우스 클릭 / J | 공격 (3연격 콤보 — 3타째는 강하고 넓은 마무리) |
| Space / Shift | 대시 (무적 프레임 0.22초, 쿨다운 1.5초) |
| M | 음소거 |
| R | (사망 시) 재도전 |

## 개발 진행 상황

- [x] **M1. 코어 전투** — 이동·3연격·대시, 적 3종(슬라임/해골 궁수/돌진 멧돼지), 타격감(히트스톱·셰이크·파티클·데미지 숫자), 웨이브 서바이벌, 절차 합성 사운드
- [ ] M2. 던전 루프 — 절차 생성 층, 방 선택 갈림길, 레벨업·특성, 층 보스
- [ ] M3. 성장의 깊이 — 5층, 특성 시너지, 유물
- [ ] M4. 메타 & 거점 — 영혼 파편, 영구 업그레이드, 직업
- [ ] M5. 폴리싱 — BGM, 층 테마, 밸런스

## 테스트 훅 (자동 검증용)

- `?autostart=1` — 시작 화면 건너뛰기
- `?demo=1` — 데모 봇 자동 플레이 (스크린샷 검증용)
- `window.Game` — 콘솔에서 게임 상태 확인 가능
