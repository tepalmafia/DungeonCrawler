# 던전 크롤러: 심연의 탑 (웹 버전)

외부 프로그램·라이브러리 없이 순수 HTML5 + Canvas + Web Audio로 만든 로그라이크 던전 크롤러 액션 RPG.
기획안: [`docs/GAME_DESIGN.md`](../docs/GAME_DESIGN.md)

## 🎮 바로 플레이

**https://tepalmafia.github.io/DungeonCrawler/** — 설치 없이 브라우저에서 바로 플레이 (main 브랜치 푸시 시 자동 배포)

## 실행 방법 (로컬)

`web/index.html`을 브라우저에서 열면 바로 실행됩니다. 또는 로컬 서버로:

```bash
cd web
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 조작

| 키 | 동작 |
|---|---|
| WASD / 방향키 | 이동 |
| 마우스 클릭 / J | 공격 (3연격 콤보 — 3타째는 강하고 넓은 마무리) |
| Space / Shift | 대시 (무적 프레임 0.22초, 쿨다운 1.5초) |
| K / 우클릭 | 직업 스킬 (검사 회전 베기 / 궁수 화살비 / 마도사 메테오) |
| 1 / 2 / 3 | (레벨업 시) 특성 카드 선택 |
| Tab | 획득 목록 (특성·유물 상세 + 현재 스탯) |
| ESC / P | 일시정지 (일시정지 중 Q — 런 포기하고 정산) |
| M | 음소거 |
| R | (사망/클리어 시) 즉시 재도전 |
| ← / → | (거점, 첫 정복 후) 열기 난이도 조절 |

## 개발 진행 상황

- [x] **M1. 코어 전투** — 이동·3연격·대시, 적 3종(슬라임/해골 궁수/돌진 멧돼지), 타격감(히트스톱·셰이크·파티클·데미지 숫자), 절차 합성 사운드
- [x] **M2. 던전 루프** — 절차 생성 방 + 갈림길 문 선택(전투/정예/보물/모닥불), XP·레벨업, 정예 변종, 상태이상, 1층 보스
- [x] **M3. 성장의 깊이** — 5층 전체(지하 묘지→곰팡이 동굴→잊힌 감옥→용암 심층→심연의 옥좌), 층별 환경 기믹(독 안개/용암/어둠), 적 12종, 보스 5종(공격 프리미티브 조합), 특성 30종(독 태그 신설), 유물 20종(커먼~레전더리 등급, 보물상자 드랍 + 보스 3택1)
- [x] **M4. 메타 & 거점** — 영혼 파편(런 정산 카운트업), 거점 화면(출발/기억의 제단/직업 선택), 영구 업그레이드 5종(육체·완력·바람·기회·수확), 직업 3종(검사 근접 3연격 / 궁수 관통 화살 / 마도사 유도 마탄+폭발), localStorage 세이브
- [x] **M5. 폴리싱** — 절차 생성 BGM 12테마(1~10층 각각 고유 곡 + 거점 + 보스 전투곡, 16분음표 시퀀서 합성 — 심층 5곡은 저음 패드가 깔린 어두운 변주), 보스 등장 위압 사운드(서브베이스 낙하 + 불협 클러스터 + 전쟁 나팔), 열기 모드 0~5(첫 정복 후 해금: 적 HP/수/속도 강화·회복 감소·보스 강화, 파편 +20%/열기), 시드 공유(?seed= 로 같은 던전 재도전), 일시정지(ESC/P), XP 커브 층 보정

- [x] **아트 리마스터** — 스프라이트 24×24 리마스터(3단계 명암), 자동 1px 아웃라인, 전체 발밑 그림자, 플레이어 걷기 프레임, 벽 오토타일(정면/윗면), 층 테마 바닥 데코 10종, 횃불 조명, 층 컬러 그레이딩 + 비네트

- [x] **확장 콘텐츠** — 던전 10층(6~10층 심층 테마 + 각성 보스), 직업 스킬 3종(K/우클릭: 회전 베기·화살비·메테오), 직업 전용 특성 9종, 보스 기믹 체크(재생/중장갑/백열/어둠 장막 — 정답 특성 트리 연구), 도감, 테스트 모드, 자동 타겟팅

- [x] **심층 전용 몬스터 5종** — 층마다 새 몬스터가 등장: 폭탄벌레(6층·자폭), 가시덩굴(7층·고정 포탑 산탄), 처형자(8층·직사각 텔레그래프 도끼), 마그마 슬라임(9층·사망 시 분열), 공허의 눈(10층·순간이동+추적탄) + 스킬별 전용 사운드, 심층 하트 드랍 감소·적 속도/정예 층별 강화

- [x] **풀런 시뮬 기반 밸런스 2차** — XP 커브 재조정(시작 36·배율 1.24 — 초반 과속 방지, 심층에서도 성장 지속), 믹서스 재생 6/s + 부하 처치 시 재생 5초 정지(컨트롤 해법), 특성 중첩 상한(곱연산 낮게·덧셈 넉넉하게, 카드에 보유 n/상한 표시), 제단 각인 2종 추가 — 환생(레벨업 카드 리롤, E키)·유산(런 시작 커먼 유물 3택1)

- [x] **몬스터 대확장 + 중간보스** — 신규 몬스터 20종(해골 병사/방패 해골/저격 해골/벌레 떼/독두꺼비/흡혈 거머리/서리 슬라임/서리 궁수/광전사/도깨비불/주술사/수정 정령/구울/뿔벌레/마력 포탑/미믹/그림자 추적자/덩치/임프/식탐귀 — 전부 고유 행동)으로 총 37종+보스 5종. **우두머리(중간보스)**: 2층부터 방마다 12% 확률로 랜덤 몬스터가 거대화(HP×7)되어 난입, **랜덤 특성 1~2개**(신속/강철/재생/유령/소환/폭발/분열) 보유 — 처치 시 하트 확정 + 영혼 파편 즉시 지급

- [x] **무한 모드 '심연 회랑'** — 10층 정복(승리 화면)에서 **C** 키로 빌드를 유지한 채 11층부터 끝없이 하강. 심층 테마·몬스터·각성 보스가 순환하며 보스 HP 층당 +15%, 적 강화 지속(정예 40%·속도 +30% 상한). 사망 시 10층 정산을 넘는 초과분만 추가 지급, 최고 기록은 도달 층수로 갱신. 보물·모닥불 문은 층당 1회로 제한, 출입문 비주얼 리뉴얼(석조 아치·포탈 발광·타입 아이콘·명판)

**전체 마일스톤 완료** — 기획안(docs/GAME_DESIGN.md)의 M1~M5 + 확장 콘텐츠가 구현되었습니다. 밸런스 기준: docs/BALANCE.md

## 테스트 모드 (콘텐츠 확인용)

거점에서 **T** 키를 누르면 테스트 모드가 켜집니다 (`?test=1`로도 가능). 화면에 단축키 안내가 표시됩니다.

| 키 | 효과 (거점/게임 공통) |
|---|---|
| O | 영혼 파편 +500 |
| I | 도감 전체 완성 |
| Y | 직업 3종 + 열기 모드 즉시 해금 |

| 키 | 효과 (게임 중) |
|---|---|
| G | 무적 토글 |
| H | 체력 전체 회복 |
| K | 현재 방 적 전멸 |
| L | 즉시 레벨업 (특성 카드) |
| U | 랜덤 유물 획득 |
| B | 현재 층 보스방 직행 |
| N | 다음 층 이동 (5층에서는 즉시 클리어) |

## 봇 모드 (자동 플레이 테스트)

`?bot=1`로 켜거나, 테스트 모드에서 **V** 키로 언제든 토글합니다 (화면 우상단에 🤖 표시).
봇은 텔레그래프 회피(자폭 반경·처형 구역·돌진 경로·충격파 링·투사체), 위험 지대 이탈(용암/독 안개/불길),
우선순위 타겟팅(영혼 구슬 > 강령술사 > 최근접), 직업별 전투(근접 콤보/원거리 카이팅), 스킬 사용,
카드 자동 선택(보유 태그 시너지 우선), 끼임 탈출까지 수행합니다.

- `?bot=1` — 봇 모드 시작 (테스트 모드 자동 활성화)
- `?botloop=1` — 사망/클리어 시 자동 재시작 (소크 테스트, 우상단에 런/승 수 표시)
- `?ff=N` — N배속 (1~8, 봇 장시간 테스트용)
- 조합 예: `?botloop=1&ff=4&class=archer&seed=TEST1`

## 테스트 훅 (자동 검증용)

- `?autostart=1` — 시작 화면 건너뛰기
- `?demo=1` — (구) 데모 봇 — 검증 스크립트 호환용, 새 검증은 `?bot=1` 권장
- `?floor=N` — N층 직행 (1~10)
- `?class=archer` — 해금된 직업으로 시작 (knight/archer/mage)
- `?seed=ABC123` — 시드 지정 런 (같은 시드 + 같은 선택 = 같은 던전)
- `?jump=boss` — 현재 층 보스방 직행 (floor와 조합 가능)
- `window.Game` — 콘솔에서 게임 상태 확인 가능
