# 던전 크롤러: 심연의 탑 (웹 버전)

외부 프로그램·라이브러리 없이 순수 HTML5 + Canvas + Web Audio로 만든 로그라이크 던전 크롤러 액션 RPG.
기획안: [`docs/GAME_DESIGN.md`](../docs/GAME_DESIGN.md)

## 🎮 바로 플레이

**https://tepalmafia.github.io/DungeonCrawler/** — 설치 없이 브라우저에서 바로 플레이 (main 브랜치 푸시 시 자동 배포)

## 실행 방법 (로컬)

`web/index.html`을 브라우저에서 열면 바로 실행됩니다. 또는 로컬 서버로:

```bash
cd web
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

## 조작

| 키 | 동작 |
|---|---|
| WASD / 방향키 | 이동 |
| 마우스 클릭 / J | 공격 (3연격 콤보 — 3타째는 강하고 넓은 마무리) |
| Space / Shift | 대시 (무적 프레임 0.22초) — **적 공격 직전에 피하면 완벽 회피**(슬로모+확정 크리), **대시 중 공격 = 파생기** |
| K / 우클릭 | 직업 스킬 (검사 회전 베기 / 궁수 화살비 / 마도사 메테오) |
| 1 / 2 / 3 | (레벨업 시) 특성 카드 선택 |
| Tab | 획득 목록 (특성·유물 상세 + 현재 스탯) |
| ESC / P | 일시정지 (일시정지 중 Q — 런 포기하고 정산) |
| M | 음소거 |
| R | (사망/클리어 시) 즉시 재도전 |
| ← / → | (거점, 첫 정복 후) 열기 난이도 조절 |

- [x] **M3. 성장의 깊이** — 5층 전체(지하 묘지→곰팡이 동굴→잊힌 감옥→용암 심층→심연의 옥좌), 층별 환경 기믹(독 안개/용암/어둠), 적 12종, 보스 5종(공격 프리미티브 조합), 특성 30종(독 태그 신설), 유물 20종(커먼~레전더리 등급, 보물상자 드랍 + 보스 3택1)

- [x] **M5. 폴리싱** — 절차 생성 BGM 12테마(1~10층 각각 고유 곡 + 거점 + 보스 전투곡, 16분음표 시퀀서 합성 — 심층 5곡은 저음 패드가 깔린 어두운 변주), 보스 등장 위압 사운드(서브베이스 낙하 + 불협 클러스터 + 전쟁 나팔), 열기 모드 0~5(첫 정복 후 해금: 적 HP/수/속도 강화·회복 감소·보스 강화, 파편 +20%/열기), 시드 공유(?seed= 로 같은 던전 재도전), 일시정지(ESC/P), XP 커브 층 보정

- [x] **풀런 시뮬 기반 밸런스 2차** — XP 커브 재조정(시작 36·배율 1.24 — 초반 과속 방지, 심층에서도 성장 지속), 믹서스 재생 6/s + 부하 처치 시 재생 5초 정지(컨트롤 해법), 특성 중첩 상한(곱연산 낮게·덧셈 넉넉하게, 카드에 보유 n/상한 표시), 제단 각인 2종 추가 — 환생(레벨업 카드 리롤, E키)·유산(런 시작 커먼 유물 3택1)

- [x] **로그라이크 심화** — **미지의 기연**(? 문, 층당 1회): 받아들이기 전엔 무엇인지 알 수 없다 — 60% 순이익(유물/공격력/최대HP/파편) · 25% 대가 있는 이익(저주 유물) · 15% 함정(HP -2). 다가가면 수락, 문으로 나가면 거절. **전설 특성 4종**(황금 카드, 극저확률): 무한의 갈망(중첩 상한 해제)·시간 왜곡(스킬 쿨 절반)·유리 영혼(공+3/HP-2)·왕의 권능(처치 시 5% 영혼 폭발). **도감 보상**: 몬스터 10/20/30/42종 발견 시 파편 +100/150/200/400

- [x] **몬스터 대확장 + 중간보스** — 신규 몬스터 20종(해골 병사/방패 해골/저격 해골/벌레 떼/독두꺼비/흡혈 거머리/서리 슬라임/서리 궁수/광전사/도깨비불/주술사/수정 정령/구울/뿔벌레/마력 포탑/미믹/그림자 추적자/덩치/임프/식탐귀 — 전부 고유 행동)으로 총 37종+보스 5종. **우두머리(중간보스)**: 2층부터 방마다 12% 확률로 랜덤 몬스터가 거대화(HP×7)되어 난입, **랜덤 특성 1~2개**(신속/강철/재생/유령/소환/폭발/분열) 보유 — 처치 시 하트 확정 + 영혼 파편 즉시 지급

- [x] **무한 모드 '심연 회랑'** — 10층 정복(승리 화면)에서 **C** 키로 빌드를 유지한 채 11층부터 끝없이 하강. 심층 테마·몬스터·각성 보스가 순환하며 보스 HP 층당 +15%, 적 강화 지속(정예 40%·속도 +30% 상한). 사망 시 10층 정산을 넘는 초과분만 추가 지급, 최고 기록은 도달 층수로 갱신. 보물·모닥불 문은 층당 1회로 제한, 출입문 비주얼 리뉴얼(석조 아치·포탈 발광·타입 아이콘·명판)

- [x] **런 곡선 개편 (R1~R3)** — **난이도 곡선 뒤집기**: 사망의 90%가 1~2층에 몰리던 역전 곡선 보정 — 1~2층 적 수 −2·2층 우두머리 완화, 대신 7층+ 정예·우두머리 접촉 피해 2 + 심층 하트 추가 감쇠 (후반에도 죽음이 가깝다). **위협 세트**: 전투방 45%는 손제작 조합 10종(방패벽 뒤 저격수 / 힐러+덩치 / 자폭+돌진 교차 등)이 무리의 뼈대 — 물량이 아니라 우선순위 퍼즐. **지름길**: 3·6층 보스 후 갈림길 — 한 층을 건너뛰지만 도착 층은 정예 2배·우두머리 강화, 대신 정예가 파편을 떨군다 (런 중반의 큰 결정)

- [x] **전투 재미 개편 (손맛 동사 확장)** — **완벽 회피**: 적 공격이 명중하기 직전 대시로 피하면 시간이 0.35배로 늘어지고 다음 일격이 확정 크리티컬 (텔레그래프가 '피할 것'에서 '노릴 것'으로). **대시 파생기**: 대시 중 공격 시 직업별 특수기 — 검사 돌진 찌르기(관통 일격) / 궁수 후퇴 사격(3발 부채꼴) / 마도사 점멸 폭발. **벽 충돌**: 마무리 일격·회전 베기의 강넉백으로 적을 벽에 처박으면 추가 피해 — 지형이 무기가 된다. **동사 특성 3종**: 지진파(마무리가 충격파 발사)·도탄(투사체 벽 반사)·불꽃 대시(대시가 화염 자취). **문 수식어**: 전투/정예 문에 35% 확률로 위험-보상 트레이드오프(사나운 무리: 적↑+파편 보너스 / 보물 지킴이: 정예 2기+상자) — 문 선택이 상태에 따라 답이 달라진다

- [x] **맵 다양화 (손제작 템플릿)** — Spelunky/Isaac 방식: 전투방이 **손제작 방 템플릿 13종**(기둥 홀·쪼개진 호수·외곽 링·좁은 목·감옥 창살·지그재그·세 갈래 회랑·벽감 2종·항아리 창고·용암 십자·쌍둥이 방책·왕좌 전정) + 좌우 반전 + 무작위 잔해(구 방식)에서 추첨된다. 테마 맞춤 가중치(용암 층은 호수형, 감옥 층은 창살형). **층 전용 템플릿 10종**: 묘지(납골당·묘비 정원) / 안개(포자 골짜기·균사 미로 — 위험 지대가 독 안개로) / 감옥(독방 블록·처형장 — 비밀 감방 포함) / 용암(용암 강·잿더미 섬) / 어둠(옥좌 길·공허 십자)은 해당 테마 층에서만 등장해 층 개성을 만든다. **항아리**(부수면 파편, 진귀한 항아리는 하트 확률)와 **균열 벽**(부수면 비밀 벽감이 열린다 — 보상 은닉)으로 방 안 탐색 보상 신설. 도달 불가 지역 자동 봉인(BFS)으로 적이 갇히는 사고 원천 차단, 벽감엔 적 스폰 금지

- [x] **로그라이크 심화 2차** (명작 분석 docs/ROGUELIKE_STUDY.md 적용) — **스킬 진화**: 직업 특성 3장 + **Lv.12**에 스킬의 형태가 바뀐다 (회전 베기→회오리 베기: 1.1초 이동 회전+추가 타격 / 화살비→화살 폭풍: 24발·범위 확대 / 메테오→쌍둥이 메테오). 3장을 먼저 모으면 '개화 대기', Lv.12 도달 순간 진화 — 봇 계측으로 시점을 런 중반(4~5층)에 맞춤. 카드에 진행도(⚡ 스킬 진화 n/3 · Lv.12) 표시. **사망 리포트**: 정산 화면에 사인(무엇에게 당했나)·지난 런과의 진전 비교 표시. **유물 시각 흔적**: 레어+ 유물은 캐릭터 주위를 도는 등급색 문장으로 표시, 광전사 투구(위기 아우라)·불사조 깃털(불씨)·유리 대검(섬광)은 전용 연출. **오늘의 탑**: 거점에서 D — 날짜 기반 고정 시드 도전, 오늘의 최고 기록·도전 횟수 저장

- [x] **보스 버스트 상한 + 연출 완화** — 보스 대상 타격당 피해 상한(최대 HP의 1.5%): 특성이 쌓인 빌드도 보스 기믹·페이즈 2를 보기 전에 녹이지 못한다 (강빌드 실측 TTK 15~57초, 약한 빌드는 영향 없음). 히트스톱을 크리티컬·굵직한 처치로 한정하고 짧게(일반 연타는 멈춤 없음), 크리 화면 섬광 빈도·강도 완화 — "탁탁 끊기는 멈춤이 렉 같다" 피드백 반영

- [x] **모닥불 양자택일 + 공급 경제 조정** — 모닥불 방에서 **휴식(HP +2)** 과 **담금질(이번 층 동안 공격력 +1)** 중 하나만 선택(하트 수급이 좋아진 뒤에도 모닥불이 의미 있는 선택지가 되도록). 하트 드랍 경제 조정: 행운·클로버 합산 배율 상한 ×3, 방당 소프트캡, 풀피 시 감쇠. 물량전 타격 렉 수정(효과음 스로틀 · 파티클 전역 상한 · 히트스톱 과열 방지)

**전체 마일스톤 완료** — 기획안(docs/GAME_DESIGN.md)의 M1~M5 + 확장 콘텐츠가 구현되었습니다. 밸런스 기준: docs/BALANCE.md

## 봇 모드 (자동 플레이 테스트)

`?bot=1`로 켜거나, 테스트 모드에서 **V** 키로 언제든 토글합니다 (화면 우상단에 🤖 표시).
봇은 텔레그래프 회피(자폭 반경·처형 구역·돌진 경로·충격파 링·투사체), 위험 지대 이탈(용암/독 안개/불길),
우선순위 타겟팅(영혼 구슬 > 강령술사 > 최근접), 직업별 전투(근접 콤보/원거리 카이팅), 스킬 사용,
카드 자동 선택(보유 태그 시너지 우선), 끼임 탈출까지 수행합니다.

- `?bot=1` — 봇 모드 시작 (테스트 모드 + **무한 부활 기본 ON**: 체력이 닳고 죽으면 풀피 부활, F로 끄기)
- **층별 사망 리포트**: 죽을 때마다 층별로 집계되어 화면 우하단에 "사망 N회 — 1층:2 3층:1" 형태로 표시.
  콘솔에서 `BotReport()`로 JSON(도달 층/시간/레벨/처치/층별 사망) 확인 가능 — 층별 난이도 측정에 사용
- **끼임 방지 AI**: 목표(적/상자/문)에 3초간 가까워지지 못하면 방의 다른 지점으로 크게 우회(탐색 모드) 후 재접근
- `?botloop=1` — 사망/클리어 시 자동 재시작 (소크 테스트, 우상단에 런/승 수 표시)
- `?ff=N` — N배속 (1~8, 봇 장시간 테스트용)
- 조합 예: `?bot=1&ff=4&class=archer&seed=TEST1` — 4배속으로 궁수 봇이 죽어가며 몇 층까지 가는지 관찰

## 테스트 훅 (자동 검증용)

- `?autostart=1` — 시작 화면 건너뛰기
- `?demo=1` — (구) 데모 봇 — 검증 스크립트 호환용, 새 검증은 `?bot=1` 권장
- `?floor=N` — N층 직행 (1~10)
- `?class=archer` — 해금된 직업으로 시작 (knight/archer/mage)
- `?seed=ABC123` — 시드 지정 런 (같은 시드 + 같은 선택 = 같은 던전)
- `?jump=boss` — 현재 층 보스방 직행 (floor와 조합 가능)
- `window.Game` — 콘솔에서 게임 상태 확인 가능
